各種ファイルの役割と構造について

** HTML file
- index.html ... login画面
- resuster.html ... 登録画面
- change.html ... 資産の変更
- search.html ... 資産の検索
- submit.html ... 資産の登録
- list.html ... 資産の一覧表示
- user.html ... ユーザ情報

** CSS file
- style.css
- フロント担当に任せます

** JS file
- root.js ... データテーブルの管理をする
- scripts.js ... ファイルを読み込み表示する

** images file
- 画像ファイル群

** data file
- data.csv ... これのみ
